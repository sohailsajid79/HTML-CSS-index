HTML Overview

Getting Started

HTML stands for HyperText Markup Language and is used to create the structure and content of a webpage. Most HTML elements contain opening and closing tags with raw text or other HTML tags between them.

Tags Description
<html> </html> All html code is nested in <html> and ends with </html>
<head> </head> Contains the meta (self-closing tag), the broswer title element <title> </title> and links to external stylesheets.
<body> </body> All visible content will be displayed inside the body element.
<h1> </h1>... Headings and sub-headings range from <h1> to </h6>, and are used to provide titles for sections of content.
<p> </p> The <p> element defines a paragraph and always starts on a new line.
<span> </span> The <span> element is an inline container and can be used to group text for styling purposes like this!
<div> </div> The <div> tag is used as a container for HTML elements - which is then styled with CSS.
<em> <em> or
<strong> <strong>		 These tags are used to emphasise text where <em> will italicise and <strong> will highlight the text as bold.
<!-- --> The comment tag is used to insert comments in the HTML source code. Comments are not displayed in the browsers.
<br> <br> is a self-closing tag and it inserts a single line break
<hr> <hr> Another self-closing tag and is used to define thematic changes in the content and is displayed as a horizontal rule. As seen below.

Semantic

These elements clearly describes its meaning to both the browser and the developer, and define different parts of the web page.

Tags Description
<header> </header> The header represents a container for introductory content.
<nav> </nav> The nav tag defines a set of navigation links.
<main> </main> This tag specifies the main content of the webpage.
<section> </section> The tag defines a section on the webpage.
<article> </article> The tag specifies independent, self-contained content.
<aside> </aside> The aside content is often placed as a sidebar on the webpage.
<footer> </footer> The tag defines a footer for the webpage and typically contains:
  • Author's information
  • Copyright information
  • Back to top links
  • Related webpages

Attributes

All HTML elements can have attributes and they provide additional information about the elements

Attribute Description
<a> </a> The href attribute requires a URL to create the hyperlink and the target value "_blank" tells the browser to open the hyperlink in a new tab
<img> The image tag can take the following attributes src="", alt="", width="", height="" and style="". The src can either be url or "./pathway/folder/to_image"
<link> Creates a connection to the external stylesheet using rel="stylesheet" href="filename.css"
<a> The href attribute of the anchor tag creates a hyperlink that can point to external webpages, files on the same server or another section of the webpage.

CSS Overview

Getting Started

Cascading Style Sheets (CSS) is the language used to style an HTML webpage and describes how HTML elements should be displayed. CSS rule consists of a selector and a declaration block.

CSS Selector Types

Attributes Description
Universal Selector * {} will target all the elements on the webpage
Type Selector Selects by the tag name h1, h2, h3 {}
class .class {} selector selects elements with a specific class attribute
id #id {} selector styles the element with the specified id

CSS Properties

Properties Description
font-family {} The property specifies the font for an element
font-size {} Sets the size of a font
font-weight {} Sets how thick or thin characters in text should be displayed
font-style {} Specifies the font style for a text
text-transform {} Controls the capitalisation of text. Some of the values include uppercase; lowercase;
text-decoration {} Decorations are added to text, and is a shorthand property for:
  • text-decoration-line:
  • text-decoration-color:
  • text-decoration-style:
  • text-decoration-thickness:
line-height {} Specifies the height of a line
letter-spacing {} Increases or decreases the space between characters in a text
word-spacing {} Increases or decreases the white space between words
text-align {} Specifies the horizontal alignment of text in an element
vertical-align {} Sets the vertical alignment of an element
